A Simple HTML Document


  • .The <!DOCTYPE html> សេចក្តីប្រកាសកំណត់ ថាឯកសារនេះគឺជាឯកសារ HTML5
  • The <html> element គឺជាធាតុឫសនៃ HTML ទំព័រ
  • The <head>ធាតុមានព័ត៌មានមេតាអំពី ទំព័រ HTML
  • The <title> eelement បញ្ជាក់ចំណងជើងសម្រាប់ ទំព័រ HTML (ដែលត្រូវបានបង្ហាញនៅក្នុងរបារចំណងជើងរបស់កម្មវិធីរុករក ឬនៅក្នុងផ្ទាំងរបស់ទំព័រ)
  • The <body> ធាតុកំណត់ document និងជាធុងសម្រាប់មាតិកាដែលអាចមើលឃើញទាំងអស់ដូចជា ចំណងជើង, កថាខណ្ឌ, រូបភាព, តំណភ្ជាប់, តារាង, បញ្ជី។ល។
  • The <h1> element កំណត់ចំណងជើងធំ
  • The <p>element កំណត់កថាខណ្ឌ


Example

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<span> Well Come to my Web <span >
<p>This is a paragraph.</p>

</body>
</html>
